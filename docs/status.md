# Service status pages

If deployments or API calls are failing, check these upstream status pages first.

- [Railway status](https://status.railway.com/)
- [OpenRouter status](https://status.openrouter.ai/)
- [GitHub status](https://www.githubstatus.com/)

If you need help, join the [PlanExe Discord](https://planexe.org/discord). Developers hang out there and can help troubleshoot issues or alert other developers about outages.
